# ToDo-Liste
Unsere Website bietet eine einfache Möglichkeit, Aufgaben zu organisieren und den Überblick zu behalten. Es können neue Todos erstellt sowie bestehende Aufgaben bearbeitet oder gelöscht werden. Zu jeder Aufgabe lassen sich zusätzliche Informationen wie Notizen, eine Kategorie oder ein Fälligkeitsdatum hinzufügen, um die Planung zu erleichtern. Zusätzlich können Aufgaben als erledigt markiert oder als wichtig hervorgehoben werden, sodass Prioritäten klar erkennbar sind. Alle Aufgaben werden gespeichert und in einer übersichtlichen Liste angezeigt. Auf der linken Seite zeigt ein Kalender an, an welchen Tagen Aufgaben geplant sind.
